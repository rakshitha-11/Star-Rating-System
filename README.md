⭐ Star Rating System

The Star Rating System is an interactive web application that allows users to rate a service from 1 to 5 stars and provide written feedback.
It features real-time star highlighting, emoji-based rating messages, and a modern responsive design.

Features
Interactive 5-Star Rating
Emoji-based Rating Messages
Hover & Click Effects
Feedback Text Area

Technologies Used
HTML5
CSS3
JavaScript
